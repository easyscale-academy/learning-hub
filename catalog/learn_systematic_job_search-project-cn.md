# Learn Systematic Job Search

## Overview

目前看来，最优秀的 Tech 学生（MIT、CMU 等顶校）的求职流程是：建一个 Excel 表，每天在招聘平台上搜一搜，一个个看过去，手动录入到表格里。这已经比大部分人做得好了——至少有追踪。

**但这仍然是一个极其低效的流程。**

我们的模块是一套 **AI 驱动的系统化求职工具链**。在精确描述简历、经历、目标、特定要求的前提下，借助 AI 的能力：

- **AI 调研工具**：生成个性化搜索策略，调研目标公司的招聘入口和 H1B 友好度
- **AI 操作浏览器**：自动化抓取招聘平台上的最新岗位
- **AI 精准定位**：找到岗位背后的面试官、HR、校友等关键联系人
- **AI 辅助触达**：用你的杀手级项目主动出击，大幅提高回复率

这是一个基于我们构建的 Job Hunting AI 工具的**系统化投递流程**，大大提升了：

| 传统方式 | 我们的方式 |
|---------|-----------|
| 每天手动搜索 | AI 自动抓取、筛选、排序 |
| 海投简历等回复 | 精准定位联系人，主动触达 |
| 不知道公司是否 sponsor | AI 调研 H1B 历史数据 |
| 项目做了没人看 | 项目直接发给对口的人 |

**这是 AI 时代，没有职业积累的学生最靠谱的求职方式。**

## 模块 1：搜索策略与目标公司

### 问题

> "我的背景能搜哪些岗位？需要哪些筛选条件？应该投哪些公司？"

### 解决方案

**AI 调研工具**——分析你的 Profile，生成：

**1. 搜索词策略**
- **L1 精准匹配**：直接匹配目标岗位名称
- **L2 技能组合**：核心技能 + 岗位类型
- **L3 宽泛搜索**：最大覆盖，不遗漏机会

**2. Filter 建议**
- 针对 LinkedIn、Indeed、Glassdoor 等平台的具体 Filter 设置
- 根据你的经验水平、签证需求、地点偏好定制

**3. 目标公司列表**
- AI 联网搜索匹配的公司
- 获取官方 Career Page 链接
- 调研 H1B Sponsor 历史

### 输出示例

```
job/query.tsv   - 搜索词列表
job/filter.tsv  - Filter 建议
job/company.tsv - 目标公司列表（含 Career Page 和 H1B 友好度）
```

## 模块 2：LinkedIn 岗位自动抓取

### 问题

> "每天在招聘网站上刷岗位太累了，而且很难判断哪些值得投。"

### 解决方案

**AI 操作你的浏览器**——根据你的个人背景和特定要求，自动完成：

1. **打开 LinkedIn Jobs**，使用你的个性化搜索词
2. **逐个点开岗位**，阅读完整 JD
3. **对比你的背景**，评估匹配度（基于你的简历、技能、目标）
4. **调研 H1B 支持**（如有签证需求）
5. **按匹配度排序**，输出 TSV 文件

### 个性化体现

- 搜索词来自你的 Profile，不是通用模板
- 匹配度评估基于你的具体技能和经历
- 排序考虑你的特定要求（如地点、远程、公司类型等）

### 输出示例

```tsv
title	company	location	posting_date	short_description	reasoning	job_link
Data Scientist	Stripe	Remote	2 days ago	Build ML models for fraud detection	技能匹配度高，支持 H1B，工程文化优秀	https://linkedin.com/jobs/view/123
ML Engineer	Notion	SF (Hybrid)	1 week ago	Optimize recommendation systems	匹配 ML 背景，独角兽公司，成长空间大	https://linkedin.com/jobs/view/456
```

### 核心价值

- **每天 5 分钟**替代 2 小时的手动搜索
- **不遗漏**符合条件的新岗位
- **有优先级**知道先投哪些

## 模块 3：定位关键联系人

### 问题

> "投了简历石沉大海，不知道谁在看我的简历，更不知道怎么 follow up。"

### 解决方案

**AI 操作你的浏览器**——针对你申请的特定岗位、特定公司，定位关键人物：

| 角色 | 价值 |
|-----|------|
| **Recruiter** | 负责筛选简历，可以帮你进入面试流程 |
| **Hiring Manager** | 最终决策者，最了解岗位需求 |
| **Future Peers** | 未来同事，可以内推、提供内部信息 |
| **Alumni** | **你的**同校校友，更愿意帮忙、回复率更高 |

### 个性化体现

- 根据**你的学校**找到在目标公司工作的校友
- 针对**你申请的具体岗位**找到对应的 Recruiter 和 Hiring Manager
- 基于**你的背景**匹配最可能帮忙的联系人

### 输出示例

```tsv
name	role	company	title	contact	reason
Jane Doe	Recruiter	Stripe	Technical Recruiter	LinkedIn: /in/janedoe	负责 Data Science 岗位招聘
John Smith	Alumni	Stripe	Senior Data Scientist	LinkedIn: /in/johnsmith	CMU 校友，同岗位，可内推
```

## 模块 4：发送 Cold Message

### 问题

> "找到人了，但不知道怎么开口，发了消息也没人回。"

### 解决方案

**AI 工具**——在我们之前做了**针对目标岗位的杀手级项目**的前提下，辅助发送 Cold Message：

**核心公式：**

```
你们招 [岗位]
我做了 [匹配项目]
链接是 [在线演示]
刚好对口，可否内推 / 给个面试机会？
```

### 为什么有效

1. **项目直接展示能力**：不是空谈，是可以点开看的
2. **精准对口**：项目内容和岗位需求高度匹配
3. **降低对方成本**：不需要猜你能不能做，看项目就知道
4. **差异化**：99% 的人只会发简历

### 回复率对比

| 方式 | 预期回复率 |
|-----|-----------|
| 海投简历 | < 5% |
| Cold Message（无项目） | 10-15% |
| **Cold Message（带杀手级项目）** | **30-50%** |

## 为什么这套流程有效

### 传统求职的痛点

| 痛点 | 后果 |
|-----|------|
| 海投简历 | 投 100 份，回复 < 5 份 |
| 不知道投什么公司 | 遗漏机会，或投不匹配的岗位 |
| 定位不到人 | 只能等系统筛选，完全被动 |
| 没有项目背书 | 和其他应届生没有差异化 |

### 我们的解决方案

| 模块 | 解决的问题 |
|-----|-----------|
| 模块 1 | **不知道投什么** → AI 调研工具生成搜索策略和公司列表 |
| 模块 2 | **每天刷岗位太累** → AI 操作你的浏览器抓取、筛选、排序 |
| 模块 3 | **定位不到人** → AI 操作你的浏览器找出 Recruiter、HM、校友 |
| 模块 4 | **没有差异化** → AI 辅助带杀手级项目主动出击 |

### 核心优势

> **模块 2 + 模块 3 本来是最麻烦、最耗时间的环节（尤其是模块 3 几乎做不到），但在 AI 操作你的浏览器的能力下变得极其简单。**

> **再配合杀手级项目（等价甚至高于目标岗位所需的综合素质）和模块 4 的主动出击，回复率大幅提升。**
