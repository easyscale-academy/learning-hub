# Structured Articulation Coach

## Overview

**传统痛点：** 你脑子里有一堆想法，但一开口就变成"意识流"——面试官困惑，同事跟不上，自己越说越乱。不是你没内容，而是表达没有结构。

**我们的方案：** 一套 AI 驱动的结构化表达训练系统。它不是"给你答案"，而是陪你练习——出题、引导思考、逐步反馈、对比复盘。让"清晰表达"从刻意练习变成本能反应。

| 维度 | 传统方式 | AI 教练方式 |
|------|---------|------------|
| 学习路径 | 看书、背模板、自己琢磨 | 互动式引导，边学边练 |
| 练习反馈 | 没人指出问题，或反馈太笼统 | 逐步反馈，用提问引导你发现问题 |
| 场景覆盖 | 只能想到几个常见场景 | 28 个职场/生活场景，随时调用 |
| 框架掌握 | 记住名字，用不出来 | 从理解到应用，完整练习闭环 |

---

## 前置准备

- 无需编程基础
- 需要能使用 AI 对话工具（如 Claude、ChatGPT 等）
- 建议准备：你想要提升表达的真实场景（面试、汇报、沟通挑战）

## 模块 1: 框架学习

### 问题

市面上讲"沟通技巧"的书很多，但大多停留在理论层面。你看完 STAR、PREP、SCQA 这些名词，合上书——然后呢？真到需要用的时候，脑子还是一片空白。

### 解决方案

AI 教练内置 18 个经过验证的表达框架，覆盖：

- **表达观点**：PREP（结论先行）
- **讲故事/汇报**：STAR（背景→任务→行动→结果）
- **说服他人**：RIDE、Elevator Pitch
- **给反馈**：SBI、BIC
- **处理冲突**：NVC、DEAR MAN
- **指挥他人**：CO-STAR（AI 时代的沟通利器）

你可以随时向 AI 教练提问，深入理解每个框架的适用场景、使用技巧、常见误区。

### 个性化体现

- 不强制按顺序学习，根据你的需求选择框架
- 可以比较不同框架（"PREP 和 STAR 有什么区别？"）
- AI 会根据你的问题深度调整讲解详细程度

## 模块 2: 场景探索

### 问题

同一个框架，在不同场景下用法可能完全不同。"面试中被问到缺点"和"给老板汇报坏消息"都需要结构化表达，但心态、策略、重点完全不同。

### 解决方案

AI 教练内置 28 个职场/生活沟通场景的深度分析，包括：

- 面试类：自我介绍、行为面试、技术面试、反问环节
- 汇报类：日常同步、项目复盘、坏消息汇报
- 协作类：需求澄清、意见分歧、跨部门沟通
- 说服类：提案推销、资源争取、价格谈判

你可以让 AI 教练拆解任何场景的底层逻辑——面试官到底在考察什么？老板听汇报时最关心什么？

### 个性化体现

- 可以描述你自己遇到的真实场景，AI 会分析适合的框架和策略
- 支持自我评估：哪些场景你已经能处理好，哪些需要重点练习

## 模块 3: 实战练习

### 问题

"懂"和"能用"之间隔着一条鸿沟。就像学投篮，看一遍教练示范就"懂了"，但要在比赛中稳定命中，得练几百次。结构化表达也是一样——没有刻意练习，知识永远停留在纸面。

### 解决方案

AI 教练提供三种练习入口：

- **从框架出发**：指定练习某个框架，AI 出不同场景的题
- **从场景出发**：描述场景，AI 推荐合适的框架
- **从真实经历出发**：讲述你的真实情况，AI 帮你重新组织表达

每次练习都走完整闭环：

```
┌─────────────────────────────────────────────────────────────┐
│                    三阶段练习闭环                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  阶段一：理解        阶段二：表达        阶段三：复盘             │
│  ┌─────────┐        ┌─────────┐        ┌─────────┐          │
│  │ 自然表达 │   →    │ 框架组织  │   →    │ 对比分析 │          │ 
│  │ 理清思路 │        │ 逐步引导  │        │ 发现差距 │          │ 
│  └─────────┘        └─────────┘        └─────────┘          │
│       ↑                  ↑                  ↓               │
│       │                  │                  │               │
│   AI 用提问         AI 逐步确认         AI 展示参考             │
│   引导思考           每步给反馈           做对比表格             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 输出示例

练习结束后，你会得到：

**你的回答（按框架步骤标注）：**
> **S**: 去年我们团队接手了一个遗留系统的迁移项目...
> **T**: 我负责设计新的数据同步方案...
> **A**: 我做了三件事：第一，梳理了所有数据依赖关系...
> **R**: 最终项目提前两周上线，零数据丢失...

**AI 参考答案（同样按框架标注）+ 逐步对比表**

**AI 反馈：**
- 亮点：Action 部分结构清晰，用"三件事"让听众有预期
- 改进建议：Result 可以补充业务影响（如节省了多少运维成本）

## 为什么这套流程有效

### 传统方式的痛点

1. **学了记不住**：看完书合上，框架名字都记不全
2. **记住用不出**：知道有 STAR，但真到面试时脑子空白
3. **没有反馈**：自己练完不知道哪里好哪里差
4. **练习孤立**：缺乏系统性，东一榔头西一棒子

### 我们的解决方案

1. **互动式学习**：通过对话理解框架，而不是单向阅读
2. **场景化应用**：每个框架都有具体场景，学完就能用
3. **即时反馈**：AI 逐步引导，每一步都有确认和反馈
4. **闭环练习**：理解→表达→复盘，形成完整学习循环

### 核心优势

| 能力 | 传统方式 | AI 教练方式 |
|------|---------|------------|
| 深度理解 | 靠自己悟 | 问到懂为止 |
| 灵活应用 | 背模板，生硬套用 | 先理清内容，再用框架组织 |
| 持续提升 | 缺乏反馈，进步慢 | 每次练习都有对比复盘 |
| 个性化 | 千人一面 | 根据你的真实场景定制练习 |

## 适用范围

**适用人群：**
- 非英语母语者：逻辑清晰可以弥补语言不精确
- 求职者：面试中的行为问题是结构化表达的主战场
- 职场人：汇报、沟通、说服都需要清晰的表达结构
- 任何想提升沟通效率的人

**适用场景：**
- 求职面试（Behavioral Interview）
- 工作汇报（日常同步、项目复盘）
- 跨部门沟通与协作
- 说服与谈判
- 给 AI 或团队成员下达清晰指令

**核心框架优先级：**
- **STAR**：面试必备，讲故事的黄金结构
- **CO-STAR**：AI 时代必备，一次性把需求说清楚
- 其他 16 个框架：按需查阅，像字典一样使用
